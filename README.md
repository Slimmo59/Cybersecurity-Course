# Cybersecurity Portfolio

This repository contains practical Python-based cybersecurity exercises developed in a controlled lab environment.

The goal of this project is to demonstrate hands-on understanding of core cybersecurity concepts such as network analysis, traffic inspection, and basic security testing techniques.

---

## 🧪 Lab Environment

The experiments were conducted in a virtualized environment including:

* Kali Linux (attacker machine)
* Lubuntu (target)
* Windows 7 (target)
* Metasploitable (vulnerable system)

---

## 🛠️ Tools Used

* Wireshark (network traffic analysis)
* Bettercap (network reconnaissance and MITM simulations)
* Python (automation and scripting)
* VirtualBox / VM environments

---

## 📌 Focus Areas

* Network traffic monitoring and analysis
* Basic attack simulation in controlled environments
* Validation of script behavior in different OS targets
* Understanding of vulnerabilities in legacy systems

---

## 📈 Purpose

This repository is part of my continuous learning path in Cybersecurity, with a focus on building practical skills relevant to SOC Analyst and Junior Cybersecurity roles.

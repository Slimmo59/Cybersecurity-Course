# 🧱 System Architecture

## Overview

This project simulates a decentralized blockchain network using multiple nodes communicating via REST APIs.

---

## Components

- Nodes: Maintain blockchain state
- Transactions: Represent real estate data
- Consensus: Longest chain rule

---

## Security Considerations

- No authentication between nodes
- Susceptible to malicious peers
- Requires secure communication (e.g., mTLS)
